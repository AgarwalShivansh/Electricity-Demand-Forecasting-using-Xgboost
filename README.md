# Electricity Demand Forecasting using XGBoost

## Overview
This project predicts electricity demand using Machine Learning. An XGBoost Regressor model is trained on historical electricity consumption data to forecast future demand.

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Model training using XGBoost
- Model evaluation
- Saved trained model for future predictions

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost



## Dataset

The project uses the `Electricity.csv` dataset containing historical electricity demand data for model training and prediction.
